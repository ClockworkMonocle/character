A simple wrapper around the HTML `<select></select>` element.

Basic usage:

```js
<Select>
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
  <option>Option 4</option>
  <option>Option 5</option>
</Select>
```

Usage with label:

```js
<Select label="Pick one">
  <option>Option 1</option>
  <option>Option 2</option>
  <option>Option 3</option>
  <option>Option 4</option>
  <option>Option 5</option>
</Select>
```
